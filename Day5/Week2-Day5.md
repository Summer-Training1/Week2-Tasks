# Week 2 - Day 5: Reflection

## Where AI Helped
- Generated the initial CustomTkinter GUI structure (widgets, layout, frame-based page switching)
- Structured the meal plan and recipe prompts to request JSON output, so responses could be parsed and used programmatically
- Explained unfamiliar Python concepts as they came up — indentation rules, dictionaries, threading, lambda closures, JSON parsing
- Suggested code review improvements and helped design intentional bugs for the debugging exercise

## Where Manual Intervention Was Necessary
- Reusing patterns from a previous project, like the save-to-file feature and Gemini API integration
- Noticed a red error indicator in VS Code on my own, before running the code or asking for help, which helped catch an indentation mistake early
- Judged that a suggested improvement wasn't valuable enough and asked for a better one a real design decision, not just accepting suggestions
- Physically tested the app at every step, catching a factual inaccuracy in an AI-provided bug description
- Made scoping decisions throughout the week deciding what to build each day, when to stop adding features
